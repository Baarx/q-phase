## Geplante Inhalte

1. **Motivation**: Warum reicht eine Excel-Tabelle irgendwann nicht mehr?

3. **Relationale Datenbanken**: Tabellen, Primärschlüssel, Fremdschlüssel
4. **SQL-Grundlagen**:
   - `SELECT`, `FROM`, `WHERE`
   - `JOIN` (zwei Tabellen verknüpfen)
   - `ORDER BY`, `GROUP BY`
5. **Praxisbeispiel**: Mini-Schuldatenbank (Schüler, Kurse, Noten)
